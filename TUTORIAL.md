## 🤖 如何获取主流大语言模型 (LLM) 的 API Key 🔑

**博文地址：** CSDN (或您指定的其他平台)

**摘要：** 本教程将指导您如何获取目前市场上最热门的、可用于开发和集成的各大AI模型的API Key，包括OpenAI、Google Gemini、Anthropic Claude、通义千问、豆包、Kimi、DeepSeek以及聚合平台OpenRouter。

**关键词：** LLM, API Key, OpenAI, Gemini, Claude, Kimi, 通义千问, 豆包, DeepSeek, OpenRouter

-----

### **引言**

随着人工智能技术的飞速发展，大语言模型 (LLM) 已成为各行各业开发新应用的核心。要将这些强大的能力集成到您自己的程序或服务中，您需要获取对应的 **API Key**。本文将详细介绍获取主流LLM API Key的步骤。

-----

### 1\. 国际主流模型

#### 1.1. OpenAI (ChatGPT, GPT-4o, GPT-4, GPT-3.5)

| 模型 | 获取难度 | 官网/平台 |
| :--- | :--- | :--- |
| **ChatGPT** | 中 (需海外环境和支付) | [platform.openai.com](https://platform.openai.com/) |

**获取步骤：**

1.  访问 **OpenAI 官方平台**：[platform.openai.com](https://platform.openai.com/)。
2.  注册/登录您的账号（通常需要海外手机号或邮箱）。
3.  **重要：** 必须为您的账户**充值**或使用**免费的赠送额度**，否则无法生成API Key。
4.  登录后，点击右上角您的头像，选择 **"API keys"** 或直接访问：[platform.openai.com/api-keys](https://platform.openai.com/api-keys)。
5.  点击 **"+ Create new secret key"**，输入名称，然后点击创建。
6.  **务必**立即复制并保存显示的 Key，因为它只会显示这一次。

#### 1.2. Google Gemini (Gemini Pro, Gemini Flash)

| 模型 | 获取难度 | 官网/平台 |
| :--- | :--- | :--- |
| **Gemini** | 易 (需Google账号) | [ai.google.dev](https://ai.google.dev/) |

**获取步骤：**

1.  访问 **Google AI Studio**：[ai.google.dev](https://ai.google.dev/)。
2.  使用您的 Google 账号登录。
3.  在左侧导航栏或主页，点击 **"Create API key in Google AI Studio"** 或访问：[ai.google.dev/gemini-api/docs/api-key](https://ai.google.dev/gemini-api/docs/api-key)。
4.  点击 **"Create API key"** 按钮，系统会立即生成 Key。
5.  复制并妥善保存您的 Key。

#### 1.3. Anthropic Claude (Claude 3 Opus/Sonnet/Haiku)

| 模型 | 获取难度 | 官网/平台 |
| :--- | :--- | :--- |
| **Claude** | 中 (需海外环境) | [console.anthropic.com](https://console.anthropic.com/) |

**获取步骤：**

1.  访问 **Anthropic 开发者控制台**：[console.anthropic.com](https://console.anthropic.com/)。
2.  注册/登录账号（可能需要海外环境和手机号验证）。
3.  登录后，点击左侧导航栏的 **"API Keys"** 或右上角的账户下拉菜单。
4.  点击 **"Create Key"** 按钮，输入 Key 的名称。
5.  复制并保存生成的 Key。

### 2\. 国内主流模型

#### 2.1. Kimi (Moonshot AI)

| 模型 | 获取难度 | 官网/平台 |
| :--- | :--- | :--- |
| **Kimi** | 易 | [platform.moonshot.cn](https://platform.moonshot.cn/) |

**获取步骤：**

1.  访问 **Kimi 开放平台**：[platform.moonshot.cn](https://platform.moonshot.cn/)。
2.  使用手机号注册/登录。
3.  登录后，点击左侧导航栏的 **"API Key"**。
4.  点击 **"新建秘钥"** 按钮。
5.  输入秘钥名称，点击确定。
6.  复制并保存新生成的 Key。

#### 2.2. 通义千问 (Qwen3)

| 模型 | 获取难度 | 官网/平台 |
| :--- | :--- | :--- |
| **Qwen3** | 易 | [dashscope.aliyun.com](https://dashscope.aliyun.com/) |

**获取步骤：**

1.  访问 **阿里云 灵积模型服务 (DashScope)** 平台：[dashscope.aliyun.com](https://dashscope.aliyun.com/)。
2.  使用**阿里云账号**登录。
3.  在控制台左侧导航栏找到 **"API Key 管理"** 或直接访问：[dashscope.console.aliyun.com/apiKey](https://dashscope.console.aliyun.com/apiKey)。
4.  点击 **"创建新的 API Key"** 按钮。
5.  输入 Key 名称，点击创建。
6.  复制并保存 Key。

#### 2.3. 豆包 (Doubao) / 火山方舟 (ByteDance)

| 模型 | 获取难度 | 官网/平台 |
| :--- | :--- | :--- |
| **Doubao** | 易 | [maas.volcengine.com](https://www.google.com/search?q=https://maas.volcengine.com/) |

**获取步骤：**

1.  访问 **火山引擎 机器学习平台 (MAAS)**：[maas.volcengine.com](https://www.google.com/search?q=https://maas.volcengine.com/)。
2.  使用**火山引擎账号**（或抖音/手机号）登录。
3.  登录后，进入 **"控制台"**。
4.  在左侧导航栏找到 **"Access Key"** (或在右上角用户菜单中找到)。
5.  点击 **"创建 Access Key"**，这将生成一组 **Access Key ID** 和 **Secret Access Key**。
6.  **注意：** 豆包 API 通常需要同时使用这两部分 Key 进行签名验证。

#### 2.4. DeepSeek (深度求索)

| 模型 | 获取难度 | 官网/平台 |
| :--- | :--- | :--- |
| **DeepSeek** | 易 | [platform.deepseek.com](https://platform.deepseek.com/) |

**获取步骤：**

1.  访问 **DeepSeek 开放平台**：[platform.deepseek.com](https://platform.deepseek.com/)。
2.  使用手机号或邮箱注册/登录。
3.  登录后，点击左侧导航栏的 **"API Key"**。
4.  点击 **"创建新秘钥"**。
5.  复制并保存生成的 Key。

### 3\. 聚合平台

#### 3.1. OpenRouter

| 模型 | 特点 | 官网/平台 |
| :--- | :--- | :--- |
| **OpenRouter** | 聚合多种模型 | [openrouter.ai](https://openrouter.ai/) |

**特点：** OpenRouter 是一个 API 聚合平台，您可以用一个 Key 调用平台上集成的数百种模型（包括部分上面提到的模型），简化了多模型切换的复杂性。

**获取步骤：**

1.  访问 **OpenRouter 官网**：[openrouter.ai](https://openrouter.ai/)。
2.  注册/登录账号。
3.  登录后，点击右上角的头像，选择 **"Keys"** 或直接访问：[openrouter.ai/keys](https://openrouter.ai/keys)。
4.  点击 **"Create New Key"**。
5.  复制并保存生成的 Key。

-----

### 💡 API Key 使用与安全提示

1.  **保密性：** API Key 相当于您的账户密码和资金，**绝对不要**将其硬编码在客户端代码中，**不要**公开分享，**不要**上传到公共代码库（如GitHub）。
2.  **环境变量：** 推荐将 API Key 设置为**系统环境变量**，并在代码中调用，这是最安全的做法。
3.  **额度管理：** 随时关注您的平台账户余额和使用额度，避免因欠费导致服务中断。
