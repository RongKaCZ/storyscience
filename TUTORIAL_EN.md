## Introduction

As AI technology advances rapidly, large language models (LLMs) have become foundational tools for building innovative applications. To integrate these powerful capabilities into your own applications or platforms, you must obtain the corresponding **API Key**.
This guide will show you how to obtain API Keys from the major LLM platforms.

## 1. International Leading Models

### 1.1 OpenAI (ChatGPT, GPT-4o, GPT-4, GPT-3.5)

| Model       | Difficulty                                  | Platform                                                     |
| ----------- | ------------------------------------------- | ------------------------------------------------------------ |
| **ChatGPT** | Medium (requires overseas access & payment) | [https://platform.openai.com/](https://platform.openai.com/) |

**Steps:**

1. Visit the OpenAI Platform:
   [https://platform.openai.com/](https://platform.openai.com/)
2. Sign up or log in (may require overseas phone/email).
3. **Important:** You must have **active credits** before creating API Keys.
4. Open the API Keys page:
   [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys)
5. Click **‚Äú+ Create new secret key‚Äù**.
6. Copy and save the key immediately.

### 1.2 Google Gemini (Gemini Pro, Gemini Flash)
